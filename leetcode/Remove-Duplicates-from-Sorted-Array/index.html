Remove Duplicates from Sorted Array

Sat, Mar 26, 2016

题目很简单,需要注意的是vector中的erase方法,会返回已删除元素的下一个位置,所以就不需要将迭代器自增了.

具体代码如下:

class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if(nums.size()<=1)
            return nums.size();
         vector<int>::iterator it;
         int pre = nums[0];
         it=nums.begin();it++;
     for(it;it!=nums.end();)
     {
         if(*it  == pre){
             it=nums.erase(it);    //删除元素，返回值指向已删除元素的下一个位置
         }
         else{
             pre = *it;
             ++it;    //指向下一个位置
         }
     }

     return nums.size();
    }
};
